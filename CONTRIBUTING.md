# Contributing

Thanks for your interest in Factorio LLM!

## Issues

Issues are welcome for:
- Bug reports
- Feature suggestions
- Questions about usage

Please check existing issues before creating a new one.

## Pull Requests

This is a personal project and I'm not actively reviewing PRs at this time. Feel free to fork and experiment, but don't expect PRs to be merged quickly (or at all).

If you want to contribute code, please open an issue first to discuss the change.

## Code of Conduct

Be respectful. That's it.
